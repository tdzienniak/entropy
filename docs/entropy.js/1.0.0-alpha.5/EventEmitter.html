EventEmitter

new EventEmitter()

Source:
  • EventEmitter.js, line 4

Very simple event emitter implementation.

Example
const ee = EventEmitter()

ee.on('myEvent', (e) => {
  console.log(e.foo);
})

ee.emit('myEvent', {
   foo: 'bar',
}) // prints `bar` in the console

off(event, fn)

Source:
  • EventEmitter.js, line 106

Removes listener for event.

Example
const myHandler = (e) => {}
ee.on('myEvent', myHandler) // handler attached
ee.off('myEvent', myHandler) // handler detached
